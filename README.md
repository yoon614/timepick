# 📅 스케줄 매칭 중심의 알바 플랫폼, 'TimePick (타임픽)'

## 1. 기획 배경 및 의도 (Project Overview)

### **1.1. 문제점 (Pain Point)**

- **비효율적인 탐색:** 기존 구인구직 앱은 '공고 중심'으로 나열되어 있어, 구직자가 일일이 공고를 클릭해 자신의 스케줄과 대조해야 하는 번거로움이 존재함. 😩
- **유동적 스케줄 관리의 어려움:** 대학생이나 프리랜서처럼 매주 또는 매일 스케줄이 변동되는 구직자의 경우, 고정된 근무 시간대의 공고는 지원하기 어려움. 📆

### **1.2. 해결책 (Solution)**

- **역발상 매칭 시스템:** 구직자가 자신의 '가능한 시간'을 먼저 등록하면, 해당 시간에 맞는 공고만 큐레이션하여 보여주는 **스케줄 중심(Time-First) 플랫폼**. 🎯
- **스마트 캘린더:** 근무 일정 등록부터 급여 계산까지 한 번에 관리하는 올인원 툴 제공. 📊

### **1.3. 타겟 사용자 (Target User)**

- 공강 시간이나 자투리 시간을 활용해 아르바이트를 하고 싶은 대학생 및 N잡러. 👩‍🎓👨‍🎓

---

## 2. 주요 기능 (Key Features)

### **2.1 스마트 스케줄 매칭 (Smart Matching)**

- **기능:** 사용자가 자신의 주간 가용 시간을 설정(드래그 또는 시간 선택)하면, 해당 시간대와 **80% 이상 일치하는 공고**를 자동으로 필터링하여 홈 화면에 노출. 🔍
- **가치:** 불필요한 공고 탐색 시간을 획기적으로 단축. ⏳

### **2.2 직관적인 근무 캘린더 (Visual Work Log)**

- **기능:** 월간 캘린더 뷰를 통해 근무 일정을 한눈에 확인. 📅
- **구현:** 날짜별 근무지가 **배지 형태**로 시각화되며, 클릭 시 상세 근무 내역 확인 가능. 🏷️
- **특징:** 제스처(스와이프)를 활용한 직관적인 삭제 기능으로 사용 편의성 강화. ✋

### **2.3 지능형 급여 계산기 (Salary Calculator)**

- **기능:** 캘린더에 입력된 데이터를 기반으로 **이번 달 예상 수령액**을 실시간으로 계산. 💰
- **상세:** 시급 입력 시 야간수당(22:00~06:00, 1.5배), 세금(3.3%) 적용 여부를 시뮬레이션하여 오차 없는 금액 제공. 📈

### **2.4 간편 근무 기록 및 수정 (Easy Edit Mode)**

- **기능:** 복잡한 입력 없이 근무지, 시간, 시급을 빠르게 입력 및 수정. ✏️
- **UX:** 모바일 환경에 최적화된 **드롭다운** 방식의 시간 선택 UI 적용. 📱
- **데이터 검증:** 필수 항목(근무지, 시간, 시급) 누락 시 즉각적인 피드백(에러 메시지)을 제공하여 정확한 정보 입력 유도. ⚠️

### **2.5 인앱 이력서 및 원클릭 지원 (In-App Resume & One-Click Apply)**

- **기능:** 기본 인적 사항 및 경력을 앱 내부에 안전하게 저장하여, 공고 지원 시 매번 새로 작성할 필요 없이 **즉시 제출** 가능. 📝

## 3. 화면 구성 (UI/UX Structure)

| 화면명 | 주요 내용 |
| --- | --- |
| 타임픽 (Home) | 나의 타임테이블 시각화 및 매칭된 추천 공고 리스트 노출. 🌟 |
| 캘린더 (Calendar) | 근무 기록 확인, 월별 급여 합계, 근무 추가/삭제. 📊 |
| 마이페이지 (My) | 프로필 수정, 이력서 관리. 👤 |

## 4. 향후 확장 계획 (Roadmap)

### **고용주(Boss) 모드 확장**

- **계정 전환:** 하나의 앱에서 '구직자'와 '구인자(사장님)' 모드 전환 기능 개발. 🔄
- **공고 등록:** 사장님이 필요한 시간대를 타임테이블에 입력하면, 해당 시간에 근무 가능한 알바생에게 **'스카우트 제안'** 발송 기능. 📩
